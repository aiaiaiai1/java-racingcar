# java-racingcar

자동차 경주 미션 저장소

## 기능 목록 정리
- ### 입력받는 기능
    - [x] 자동차 이름을 입력받는 기능
        - 예외처리
            - [x] 쉼표를 기준으로 구분하여 입력을 받아야 한다.
    -  [x] 자동차 이동 시도횟수를 입력받는 기능
        - 예외처리
          - [ ] 최대 시도 횟수를 1~100회로 제한
- ### 출력하는 기능
    -  [x] 실행결과 문구를 출력하는 기능
    -  [x] 자동차별 이동 횟수를 출력하는 기능
    -  [x] 최종 우승자를 출력하는 기능

- ### 메인 기능
    - 자동차 경주 게임
      - [x] 전체 자동차를 포함한다.
    
    - 자동차
        -  [x] 자동차는 이름을 가진다.
        -  [x] 랜덤 값을 받아 전진 혹은 위치 유지

        -  예외처리
            -  [x] 자동차의 이름은 1~5자만 가능하다.
    - 경주 결과
      -  [x] 최종 우승 자동차
      -  [x] 경주 기록 결과
  
    - [x] 0~9사이의 랜덤 값을 생성하는 기능
